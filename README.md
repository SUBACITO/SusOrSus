Sus Or Sus
Game Name is an exciting multiplayer game developed using Unity and Photon PUN. The game offers a dynamic experience where players can perform tasks, identify imposters, and win by either completing tasks or eliminating all imposters. With a blend of strategy, teamwork, and thrilling gameplay, Game Name guarantees endless fun!

Features
Multiplayer gameplay using Photon PUN.
Interactive tasks for players.
Imposter and Crewmate roles with unique abilities.
Real-time voting and decision-making mechanics.
Customizable game options and settings.
Voice chat using Photon Voice.
Smooth animations and effects.
